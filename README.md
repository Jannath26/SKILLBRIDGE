# SkillBridge 🌉

SkillBridge is a mentorship and skill exchange platform that connects learners with mentors to make learning more collaborative, guided, and accessible.

## 🚀 Features

### 🏠 Dashboard
The dashboard provides a quick overview of the user’s mentorship activity.

- View mentorship connections
- Become a mentor by sharing your skills
- Discover recommended mentors
- Credits and active roadmap sections are available in the interface

---

### 💬 Messages
A real-time messaging system that allows users to communicate directly.

- One-to-one messaging
- Seamless mentor-learner communication
- Realtime interaction using Socket.io

---

### 👨‍🏫 Find Mentors
Users can discover mentors based on skills and interests.

- Browse available mentors
- Send mentorship session requests
- Message mentors directly
- Receive notifications when requests are sent

---

### 👤 Profile
Each user has a customizable profile containing:

- Name and title
- Role selection (Learner / Mentor)
- Skills offered
- Connected social profiles:
  - GitHub
  - LinkedIn
  - X (Twitter)

---
